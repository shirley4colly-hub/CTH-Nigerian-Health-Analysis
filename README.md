# CTH-Nigerian-Health-Analysis

## Project Overview
This project was developed as part of the **CTH Power BI Mentorship Program**.  
The goal is to analyze patient disease records of Nigerians and deliver actionable insights through an interactive Power BI dashboard.

The analysis focuses on disease patterns, death rates, demographic vulnerability, and geographic hotspots to support data-driven healthcare decisions.

## Project Objectives

- Identify incidence of **Malaria and Cholera**
- Detect and compare **death rates in 2010 and 2018**
- Determine settlements with the **highest disease occurrence**
- Analyze **age group susceptibility**
- Build an interactive storytelling dashboard
- Provide actionable health recommendations

## Tools & Technologies

- **Power BI Desktop**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- Data Visualization & Storytelling

## Key Insights

- Malaria and Cholera remain dominant infectious diseases.
- Disease burden is concentrated in specific geographic hotspots.
- Adults show higher susceptibility to diarrhoeal infections.
- Certain years recorded noticeable outbreak spikes.
- Targeted interventions can significantly reduce mortality.

## Recommendations

- Strengthen malaria and cholera control programs.
- Improve water sanitation in high-risk settlements.
- Launch adult-focused hygiene awareness campaigns.
- Enhance outbreak early warning surveillance.
- Improve early diagnosis and emergency response systems.

## Dashboard Features

- KPI cards for key health metrics  
- Disease distribution comparison (2010 vs 2018)  
- Geographic hotspot map  
- Settlement-level disease analysis  
- Age group susceptibility visualization  
- Interactive slicers for dynamic filtering  

## How to Use

1. Download the `.pbix` file from this repository.
2. Open using **Power BI Desktop**.
3. Use slicers to explore disease patterns by year, state, and gender.
